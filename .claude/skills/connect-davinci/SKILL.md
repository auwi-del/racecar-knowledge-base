## 任务

连接到 davinci-mini ROS 开发板并验证连接。

## 固定参数

| 参数 | 值 |
|------|-----|
| 开发板 IP | 192.168.5.100 |
| 用户名 | davinci-mini |
| SSH 方式 | 密钥认证（免密）✅ 已配置 |
| SSH 密码 | Mind@123（备用）|

## 执行步骤

### 1. 检查网络连通性

```bash
ping -n 2 -w 1000 192.168.5.100
```

### 2. 根据结果分支处理

#### 如果 Ping 通（0% 丢失）

执行 SSH 连接并列出 home 目录：

```bash
ssh davinci-mini@192.168.5.100 "ls -la ~"
```

确认 SSH 成功后，向用户报告：
- 连接成功 ✅
- 设备主机名和 home 目录内容摘要（列出目录下的主要文件夹和文件）

#### 如果 Ping 不通（100% 丢失）

检查本机网络接口：

```bash
ipconfig
```

分析原因并告知用户：
- 检查本机是否在 `192.168.5.x` 网段
- 列出本机当前所有活跃网卡的 IP 地址
- 建议用户将以太网线接好后，将对应网卡 IP 手动设为 `192.168.5.x`（例如 `192.168.5.2`）
- 如果本机 Wi-Fi 已在 `192.168.5.x` 段但 Ping 不通，提示检查开发板是否开机、网线是否插好
