# 多点导航（航点循环）

## 概述

多点导航基于 `nav2_waypoint_cycle` 包，让机器人依次访问用户在 RViz 中设置的多个航点，到达所有点后自动循环。

## 前置条件

已有一张建好的地图 `ai_map.yaml`，位于 `~/racecar/src/racecar/map/`。

如果没有地图，先建图：

```bash
# 终端1
ssh davinci-mini@192.168.5.100
bash ~/racecar/gmapping.sh

# 终端2
ssh davinci-mini@192.168.5.100
source ~/racecar/install/setup.bash
ros2 run racecar racecar_teleop

# 建完保存
bash ~/racecar/save.sh
```

## 启动命令

只需要一个终端：

```bash
ssh davinci-mini@192.168.5.100
bash ~/racecar/nav.sh
```

`nav.sh` 自动依次启动：
1. `Run_car.launch.py` — 底盘 + LiDAR + IMU + 编码器 + EKF + 底盘驱动
2. 等待 10 秒
3. `Run_nav.launch.py` — waypoint_cycle + Nav2 全套（AMCL + planner + controller + bt_navigator）

## 操作步骤

### 第1步：确认机器人起点

机器人默认初始位姿为 `(0, 0, 0)`（由 `nav.yaml` 中 `set_initial_pose: True` 自动设置）。不需要在 RViz 中再设 "2D Pose Estimate"，因为配置文件中已自动初始化。

### 第2步：在 RViz 中设置航点

1. 点击 **"Publish Point"** 按钮
2. 在地图上依次点击多个目标点
3. 每点一个，RViz 中会显示带编号的箭头标记

### 第3步：自动执行

- `waypoint_cycle` 自动将第一个航点通过 `/goal_pose` 发给 Nav2
- 机器人导航到该点
- 到达后自动发下一个
- 全部到达后从头开始循环

## waypoint_cycle 工作流程

```
用户在 RViz 点 "Publish Point"
       ↓
  /clicked_point (PointStamped)
       ↓
  waypoint_cycle 记录并显示带编号的箭头标记
       ↓
  自动发布 /goal_pose → Nav2
       ↓
  监听 navigate_to_pose/_action/status
       ↓
  ┌─ 到达成功 → 发下一个航点
  └─ 到达失败 → 重试一次，再失败则跳过
       ↓
  所有航点完成 → 自动从头循环
```

## 同时录制视频

多点导航 + 视频录制不冲突（`ros2 bag record` 只订阅话题，不操作硬件）：

```bash
# 新开终端
ssh davinci-mini@192.168.5.100
source /opt/ros/humble/setup.bash

# 录制全程
timeout 300 ros2 bag record /image_raw -o ~/race_video
```

## 关键参数

位于 `~/racecar/src/racecar/config/nav.yaml`：

| 参数 | 默认值 | 说明 |
|------|--------|------|
| `set_initial_pose` | True | 自动设置初始位姿（无需手动） |
| `initial_pose.x/y/yaw` | 0, 0, 0 | 默认起点坐标 |
| `desired_linear_vel` | 0.33 m/s | 导航行驶速度 |
| `goal_reached_tol` | 0.6 m | 到达目标容差 |
| `waypoint_pause_duration` | 200 ms | 每点停留时间 |
| `stop_on_failure` | False | 失败后跳过继续 |

## 命令速查

| 命令 | 作用 |
|------|------|
| `bash ~/racecar/nav.sh` | 一键启动导航+航点循环 |
| `bash ~/racecar/gmapping.sh` | 建图模式 |
| `bash ~/racecar/save.sh` | 保存地图 |
| `ros2 run racecar racecar_teleop` | 键盘遥控 |
| `ros2 topic pub /goal_pose ...` | 命令行发布导航目标 |
| `ros2 bag record /image_raw` | 录制视频 |
