# 系统概览

## 基本信息

| 项目 | 值 |
|------|-----|
| 主机名 | davinci-mini |
| 架构 | aarch64 (ARM64) |
| 内核 | Linux 5.10.0+ |
| OS | Ubuntu 22.04.5 LTS (Jammy Jellyfish) |
| ROS2 发行版 | Humble Hawksbill |
| Python | 3.10.12 |
| ROS2 中间件 | FastDDS (rmw_fastrtps_cpp) |

## 硬件配置

### 计算平台
- ARM64 开发板（Orin NX 或类似 NVIDIA 平台）
- 运行 ROS2 Humble

### 传感器和执行器

| 设备 | 类型 | 连接方式 | 设备路径 | 波特率 |
|------|------|----------|----------|--------|
| LS LIDAR M10 | 激光雷达 | 串口 | `/dev/laser` | - |
| HIPnuc IMU | 九轴惯性测量单元 | 串口 | `/dev/imu` | 115200 |
| 编码器 (磁编码器) | 轮速传感器 | 串口 | `/dev/encoder` | 57600 |
| USB 摄像头 | 视觉 | USB | `/dev/video0` | - |
| 底盘控制器 | 下位机 (PWM) | 串口 | `/dev/car` | 38400 |

### 机器人结构参数
- 车轮构型：**差速轮**
- 机器人轮廓（footprint）：`[-0.32, -0.18], [-0.32, 0.18], [0.32, 0.18], [0.32, -0.18]`（矩形，长 0.64m × 宽 0.36m）
- 机器人半径（圆形近似）：**0.60m**
- 最小转弯半径：**0.6m**

### 坐标系定义

| Frame ID | 说明 | 相对位姿（相对 base_footprint） |
|----------|------|--------------------------------|
| `base_footprint` | 机器人中心（地面投影） | 原点 |
| `base_link` | 机器人本体 | z=0.15m |
| `laser_link` | 激光雷达 | x=0.07m |
| `IMU_link` | IMU 安装位置 | x=0.1653m |
| `odom_combined` | EKF 里程计坐标系 | world frame |
| `map` | 全局地图坐标系 | world frame（仅导航时） |

## ROS2 环境变量

```
ROS_VERSION=2
ROS_PYTHON_VERSION=3
ROS_DISTRO=humble
AMENT_PREFIX_PATH=/opt/ros/humble
ROS_LOCALHOST_ONLY=0
```

## 已安装的重要系统包

系统级 ROS2 包（通过 apt 安装的二进制包）包括但不限于：

- **nav2_** 系列：amcl, behavior_tree, behaviors, bt_navigator, controller, costmap_2d, lifecycle_manager, map_server, planner, smac_planner, velocity_smoother, waypoint_follower 等
- **robot_localization**：EKF/UKF 状态估计
- **slam_toolbox**：SLAM 工具
- **slam_gmapping**：Gmapping SLAM
- **teleop_twist_keyboard / teleop_twist_joy**：遥控
- **rviz2**：可视化
- **rqt_** 系列：rqt_graph, rqt_console, rqt_reconfigure 等
- **robot_state_publisher / joint_state_publisher**：运动学
- **usb_cam**：USB 摄像头驱动
- **tf2_** 系列：坐标变换
- **rosbag2**：数据录制与回放

## ROS_DOMAIN_ID

未显式设置，使用默认值 0。如果同一网络中多个 ROS2 系统共存，需通过设置 `export ROS_DOMAIN_ID=<unique_id>` 避免串扰。
